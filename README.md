COVID-19 Data Analysis & Visualization

A Data Analytics project focused on analyzing and visualizing COVID-19 pandemic data using Python and data analysis libraries. This project helps in understanding infection trends, recovery rates, death statistics, and regional comparisons through exploratory data analysis and visualizations.

📌 Project Overview

The COVID-19 pandemic generated a massive amount of public health data.
This project aims to:

Analyze COVID-19 datasets
Identify trends and patterns
Perform exploratory data analysis (EDA)
Generate meaningful visualizations
Compare cases, deaths, and recoveries across regions
Provide data-driven insights
🚀 Features
Data Cleaning and Preprocessing
Exploratory Data Analysis (EDA)
Trend Analysis
Statistical Insights
Data Visualization using charts and graphs
Regional and Time-based Analysis
Automated Testing Structure

⚙️ Installation

Clone the repository:

git clone <your-repository-url>

Navigate to the project directory:

cd COVID19

Install dependencies:

pip install -r requirements.txt

Run Jupyter Notebook:

jupyter notebook
▶️ Usage
Add datasets inside the data/ directory
Open notebooks from the notebooks/ folder
Run the analysis cells
View generated visualizations
